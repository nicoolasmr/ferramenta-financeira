# A Única Maneira de Vencer a Inflação

Se você vender apenas planos fixos e nunca aumentar o preço, seu negócio encolhe (inflação).
O antídoto é **Expansion Revenue**.
É vender mais para quem já comprou.

## Vetores de Expansão

1.  **Assentos (Seats):** O mais comum. Empresa cresce, contrata mais, paga mais. (Ex: Slack).
2.  **Uso (Usage):** Paga por volume. (Ex: AWS, Twilio, RevenueOS).
    *   É o mais justo. Se o cliente fatura mais, você ganha mais. Alinhamento perfeito.
3.  **Features (Add-ons):** "Módulo Fiscal + R$ 50/mês".
4.  **Edições (Tiers):** Upgrade do Basic para o Pro.

## O Momento Certo (Contextual Upsell)

Não ligue para o cliente oferecendo upgrade do nada.
Ofereça quando ele bater no teto.
*   Usuário tentou exportar CSV (Feature Pro).
*   Modal: *"Essa é uma feature Pro. Quer testar por 7 dias grátis?"*
Isso converte 10x mais que e-mail frio.

## Net Dollar Retention (NDR)

Se o Expansion Revenue for maior que o Churn Revenue, seu NDR é > 100%.
Isso é **Crescimento Negativo de Churn**. O Santo Graal do SaaS.
Significa que mesmo se parar de vender, você cresce.
