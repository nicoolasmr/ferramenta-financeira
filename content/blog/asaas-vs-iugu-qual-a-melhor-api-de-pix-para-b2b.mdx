# A Batalha dos Gateways: Quem tem a melhor API?

Escolher um gateway é como casar. É difícil de sair depois.
Se a API for ruim, seu time de engenharia vai sofrer para sempre.

Analisamos tecnicamente os 3 maiores players.

## 1. Stripe (O Padrão Global)
*   **DX:** Nota 10/10. Documentação perfeita, SDKs tipados.
*   **Recursos:** Webhooks confiáveis, Sandbox ambiente idêntico ao prod.
*   **Problema:** Taxas altas no Brasil e Pix com conciliação chata.
*   **Veredito:** Melhor para quem vender globalmente.

## 2. Asaas (O Gigante do Pix)
*   **DX:** Nota 8/10. API REST limpa, melhorou muito em 2025.
*   **Recursos:** Automação de cobrança nativa (SMS/WPP).
*   **Problema:** Rate limits agressivos em contas novas.
*   **Veredito:** Melhor para B2B Nacional (focado em Boleto/Pix).

## 3. Iugu (O Veterano B2B)
*   **DX:** Nota 7/10. Funcional, mas alguns endpoints são legados.
*   **Recursos:** Split de pagamento muito robusto para marketplaces.
*   **Problema:** Documentação às vezes desatualizada.
*   **Veredito:** Sólido para Marketplaces complexos.
