# O erro da média

A fórmula clássica `LTV = ARPU / Churn` assume que a taxa de churn é constante para sempre.
Ela não é.
Clientes antigos tendem a ser mais fiéis (Churn cai com o tempo).
A fórmula clássica subestima o LTV de empresas maduras.

## Análise de Cohort (Safra) e Retenção

Para calcular certo, você precisa projetar a **Curva de Retenção**.
Mês 1: 100%
Mês 6: 60%
Mês 12: 40%
Mês 24: 35% (Estabilizou).

O LTV é a área sob essa curva, multiplicada pela margem de contribuição.

## Implementation em Python/Pandas

```python
from lifelines import KaplanMeierFitter

kmf = KaplanMeierFitter()
kmf.fit(durations, event_observed)
kmf.plot_survival_function()
```

Isso gera o gráfico de sobrevivência real da sua base.
