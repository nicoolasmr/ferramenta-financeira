# O Roteiro da Demo Perfeita

Não comece pelo começo (Login). Comece pelo Fim (O Resultado).
Mostre o Dashboard Lindo com os gráficos subindo.
*"É aqui que você quer chegar, certo?"*
O cliente concorda.
*"Agora vou te mostrar os 3 passos simples pra chegar aqui."*

## Foco na Dor

Se a dor do cliente é "Conciliação", mostre SÓ A CONCILIAÇÃO.
Não mostre o módulo de E-mail Marketing. Isso é ruído. Confunde.
Demo boa é demo curta e cirúrgica.
