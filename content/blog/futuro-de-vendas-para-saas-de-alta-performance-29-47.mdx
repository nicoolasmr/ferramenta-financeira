# O Melhor dos Dois Mundos

PLG puro tem teto (difícil fechar contratos de R$ 100k via cartão no site).
Vendas puras têm CAC alto.

PLS é usar o produto para gerar leads para vendas.
1.  Funcionários da Empresa X usam a versão Grátis.
2.  O Vendedor recebe alerta: "50 usuários ativos no domínio @cocacola.com".
3.  O Vendedor liga para o Diretor de TI: "Vi que seu time já ama nossa ferramenta. Vamos consolidar num plano Enterprise com SSO?"

A venda já está 80% feita.
