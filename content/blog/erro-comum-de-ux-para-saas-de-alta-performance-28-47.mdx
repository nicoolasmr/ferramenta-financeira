# O problema da "Tela em Branco"

O usuário cria a conta. Entra no Dashboard.
Tudo está vazio. "Nenhuma transação encontrada".
Ele não sabe o que fazer. Ele vai embora.

## Empty States Educativos

Não diga "Nada aqui". Diga:
*"Você ainda não tem transações. Que tal criar a primeira?"*
[Botão Grande: Criar Transação]

## O Checklist de Ativação

Coloque uma barra de progresso: "Setup da Conta (30%)".
1.  [x] Criar Conta
2.  [ ] Conectar Banco
3.  [ ] Emitir Nota Fiscal

Humanos têm compulsão por completar listas. Use isso a seu favor.
O "Aha Moment" deve acontecer antes do passo 3.
