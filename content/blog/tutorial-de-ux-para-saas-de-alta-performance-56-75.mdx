# Data Density vs Clareza

O CFO quer ver 50 colunas na tabela. O CEO quer ver 1 número grande.
Como desenhar para os dois?

## 1. O Princípio "Overview First, Zoom Filter, Details on Demand"

*   **Topo:** KPIs agregados (Receita Total).
*   **Meio:** Gráfico de tendência (Evolução 12 meses).
*   **Fundo:** Tabela detalhada com busca e filtros.

## 2. Alinhamento Numérico

Nunca centralize números em tabelas.
Sempre alinhe à **Direita**. E use fontes monoespaçadas (tabelares) para que os dígitos se alinhem verticalmente.
Isso permite comparar magnitudes rapidamente (`1.000,00` vs `  100,00`).

## 3. Cores Semânticas

Verde é bom? Nem sempre. (Despesa subiu = Verde?).
Em finanças, prefira cores neutras para estrutura e cores de "Alerta" (Vermelho/Laranja) apenas para anomalias que exigem atenção.
