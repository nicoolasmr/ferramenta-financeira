# A Mentira do Pipeline

Seu VP de Vendas diz: "Temos R$ 3 milhões no pipeline, vamos bater a meta de R$ 1 milhão fácil".
No final do mês, vocês fecham R$ 600k.
O que aconteceu? **Pipeline Bloated (Inchado).**

Vendedores são otimistas. Eles deixam oportunidades mortas como "Em Negociação" para não parecerem incompetentes.

## Forecast Ponderado (Weighted Forecast)

Não some o valor total. Multiplique pela probabilidade da etapa.
*   Discovery (10%): R$ 100k -> R$ 10k.
*   Demo (25%): R$ 100k -> R$ 25k.
*   Proposta (60%): R$ 100k -> R$ 60k.
*   Contrato Enviado (90%): R$ 100k -> R$ 90k.

Sua previsão real é a soma ponderada.

## Forecast Categorizado (Commit vs Best Case)

Exija que o vendedor classifique a oportunidade semanalmente:
1.  **Commit:** "Eu aposto meu emprego que isso fecha este mês."
2.  **Best Case:** "Se tudo der certo (jurídico aprovar rápido), fecha."
3.  **Pipeline:** "Tem chance, mas provavelmente fica pro mês que vem."

O CFO deve montar o budget baseado apenas no **Commit**.
