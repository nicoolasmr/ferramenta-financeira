# Growth 2.0: Dados Preditivos

Marketing tradicional é reativo. "O cliente saiu, vamos mandar e-mail de reconquista".
Marketing com IA é proativo. "O padrão de uso deste cliente sugere 80% de chance de churn em 30 dias. Vamos agir agora."

## Casos de Uso Reais

1.  **Lead Scoring Preditivo:** Analise não só o cargo (CEO), mas o comportamento no trial. Se ele instalou o script JS, a chance de conversão sobe 500%.
2.  **Oferta Dinâmica:** Se o cliente tem perfil de Enterprise, o site esconde o preço e mostra "Falar com Vendas". Se é SMB, mostra "R$ 99/mês".
3.  **Support Triage:** IA classifica tickets. Se um cliente "High LTV" abre ticket reclamando de bug crítico, ele fura a fila.
