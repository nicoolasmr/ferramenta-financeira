# O Prazer de Programar

## Stripe
A API da Stripe é o estado da arte.
*   **Cliff Time:** Zero. Você lê e entende.
*   **SDK:** Tipagem TypeScript perfeita. Autocomplete funciona.
*   **CLI:** O `stripe listen` para webhooks locais é mágico.

## Asaas
A API v3 é muito boa (RESTful padrão).
*   **Sandbox:** Funcional, mas as vezes instável.
*   **SDK:** Oficial é básico, comunidades mantém wrappers melhores.
*   **Webhooks:** Robusto, mas sem CLI oficial para debug local (use Ngrok).

## Conclusão Final

A Stripe vence em DX pura. É uma ferramenta feita por devs, para devs.
O Asaas vence em "Product Fit" para o mercado brasileiro.
Como CTO, você deve pesar se prefere DX perfeita ou Taxas melhores no Pix.
