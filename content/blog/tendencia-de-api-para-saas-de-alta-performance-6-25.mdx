# Pricing não é Matemática, é Psicologia

Você define o preço do seu SaaS baseado nos custos de servidor? Errado.
O preço é baseado no **Valor Percebido**.

## 1. O Efeito Isca (Decoy Effect)

A clássica estratégia do cinema (Pipoca P, M, G).
*   Plano Basic: R$ 100 (Poucos recursos).
*   Plano Pro: R$ 180 (Tudo o que você quer).
*   Plano Enterprise: R$ 900 (Só para âncora).

O Enterprise existe apenas para fazer o Pro parecer barato. Ninguém compra o Enterprise pelo site, mas ele vende o Pro.

## 2. Aversão à Perda (Trial Reverso)

Ao invés de dar um "Free Tier" limitado, dê o "Pro Trial" por 14 dias.
Quando o trial acaba, o usuário sente a **dor de perder** as features Pro.
A conversão de Trial Reverso para Pago é 30% maior que de Freemium para Pago.

## 3. Preços Charmosos (Charm Pricing)

Terminar em 9 (R$ 99) vs 0 (R$ 100).
Em B2B Enterprise, evite isso. Preços redondos (R$ 2.000) passam imagem de qualidade/premium.
Em B2B SMB/Prosumer, use os 9s.
