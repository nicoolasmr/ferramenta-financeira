# A Falácia do Input Manual

Você pede para o vendedor preencher:
*   Setor da Empresa
*   Receita Anual
*   Número de Funcionários

Ele chuta. Ou deixa em branco.
Aí o Marketing tenta segmentar campanha e não consegue.

## Enrichment Automático

O vendedor deve preencher apenas o **E-mail** ou **Domínio**.
O sistema deve preencher o resto via API (Clearbit, Apollo, RevenueOS).
Isso garante dados padronizados ("Technology" vs "Tech" vs "TI").

Dados limpos = Relatórios confiáveis.
