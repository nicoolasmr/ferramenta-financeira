# O Pixel morreu

Com o iOS 14+ e ad blockers, o pixel do Facebook perde 30-50% das conversões.
O Facebook reporta CAC de R$ 50.
Seu banco diz que o CAC real é R$ 100.

## Conversion API (CAPI)

A solução é enviar eventos **Server-side**.
Quando o pagamento é confirmado no RevenueOS (Backend), nós enviamos um evento autenticado para o Facebook/Google:
*"O usuário com e-mail hashed X comprou R$ 200."*

Isso recupera a visibilidade.

## Janela de Atribuição

Cuidado com a janela "View-Through". O Facebook conta conversão se a pessoa *viu* o anúncio e comprou depois (mesmo sem clicar).
Isso infla os números.
No RevenueOS, focamos em **Click-Through** hard conversions. Dinheiro na conta.
