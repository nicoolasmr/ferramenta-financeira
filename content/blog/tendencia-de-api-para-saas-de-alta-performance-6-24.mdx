# PLG: Chega de "Falar com Vendas"

Se o seu software exige uma reunião de 45 minutos para o cliente ver a cor do dashboard, você está em 2015.
Em 2026, o comprador B2B quer **Self-Service**.
Ele quer entrar, testar, ver valor (Aha Moment) e passar o cartão.

Isso é Product-Led Growth.

## Os Pilares do PLG

1.  **Barreira de Entrada Zero:** Free Trial ou Freemium sem cartão de crédito.
2.  **Time-to-Value (TTV) Curto:** O usuário deve resolver o problema dele em < 10 minutos.
3.  **Viralidade Inerente:** O uso do produto convida outros usuários (Ex: Slack, RevenueOS Invites).

## O Funil PLG

Esqueça o MQL (Marketing Qualified Lead). Focamos no PQL (Product Qualified Lead).
*   **Sign-up:** Usuário criou conta.
*   **Activated:** Usuário realizou a "Core Action" (Ex: Emitiu 1ª Nota Fiscal).
*   **PQL:** Usuário atingiu limite do plano grátis ou usou feature avançada. AGORA o time de vendas liga.
