# Além da Regressão Linear

O Excel faz uma linha reta baseada no passado.

*   "O mercado de tech está em crise."
*   "Vocês acabaram de lançar uma feature matadora."
*   "Dezembro é mês fraco para B2B."

Ela combina esses fatores qualitativos com os dados quantitativos para gerar cenários: Pessimista, Realista, Otimista.
