# A Guerra das Linguagens

## Go (Golang)
*   **Prós:** Compila muito rápido. Sintaxe simples (fácil contratar). Concorrência (Goroutines) incrível.
*   **Contras:** Garbage Collector (pode causar pausas de milissegundos). Sistema de tipos simples demais (pode causar erros de runtime).
*   **Veredito:** Ótimo para Microsserviços HTTP e APIs de rede.

## Rust
*   **Prós:** Memory Safety sem Garbage Collector. Sistema de Tipos (Algebraic Data Types) impede estados inválidos. Performance de C++.
*   **Contras:** Curva de aprendizado brutal. Compilação lenta.
*   **Veredito:** Perfeito para o "Core" (Engine de Matching, Ledger, Criptografia).

No RevenueOS, usamos **Typescript** para a API (DX) e **Rust** para o Core Ledger (Segurança).
