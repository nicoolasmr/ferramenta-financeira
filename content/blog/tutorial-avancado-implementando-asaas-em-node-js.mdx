# O Padrão Brasileiro

O Asaas é a API mais amigável para boleto/Pix no Brasil.

## 1. Setup do Cliente (`axios`)

```javascript
const asaas = axios.create({
  baseURL: 'https://api.asaas.com/v3',
  headers: { access_token: process.env.ASAAS_KEY }
});
```

## 2. Criando o Customer

É mandatório para emitir boleto registrado. Use CPF/CNPJ válidos na Sandbox.

```javascript
const { data: customer } = await asaas.post('/customers', {
  name: 'João Silva',
  cpfCnpj: '...',
  email: '...'
});
```

## 3. Dealing com Webhooks

O Asaas envia um array de eventos. Nem todo mundo trata isso.

```javascript
// Express Handler
app.post('/asaas-webhook', (req, res) => {
  const { event, payment } = req.body;
  if (event === 'PAYMENT_RECEIVED') {
    // Liberar acesso
  }
  res.json({ received: true });
});
```

## Segurança

Sempre valide o header `asaas-access-token` do webhook para garantir que foi o Asaas que enviou.
